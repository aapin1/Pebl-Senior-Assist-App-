# Pebl Tech Help Website

A modern, responsive website for Pebl Tech Help - an AI-powered voice assistant app designed to help seniors with technology questions.

**Live at:** https://peblapp.help

## 🌟 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Accessibility First**: High contrast support, keyboard navigation, screen reader friendly
- **Modern UI**: Clean, professional design with smooth animations
- **SEO Optimized**: Proper meta tags, structured data, and semantic HTML
- **Fast Loading**: Optimized images, efficient CSS, and minimal JavaScript
- **Progressive Enhancement**: Works even with JavaScript disabled

## 🚀 Getting Started

### Local Development

1. Clone or download the website files
2. Open `index.html` in your web browser
3. For development, use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

### Deployment Options

#### Option 1: Netlify (Recommended)
1. Create a Netlify account
2. Drag and drop the website folder to Netlify
3. Your site will be live instantly with a custom URL
4. Connect your domain when ready

#### Option 2: Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the website directory
3. Follow the prompts to deploy

#### Option 3: GitHub Pages
1. Create a GitHub repository
2. Upload the website files
3. Enable GitHub Pages in repository settings
4. Your site will be available at `username.github.io/repository-name`

#### Option 4: Traditional Web Hosting
1. Upload files via FTP to your web host
2. Ensure `index.html` is in the root directory
3. Configure your domain to point to the hosting

### Content Updates
- Edit `index.html` for main content
- Update `faq.html` for questions and answers
- Modify `css/style.css` for styling changes
- Add functionality in `js/script.js`

### Adding New Pages
1. Create a new HTML file (e.g., `about.html`)
2. Copy the navigation and footer from `index.html`
3. Add your content in between
4. Update navigation links in all pages
